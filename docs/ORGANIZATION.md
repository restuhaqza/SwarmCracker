# Documentation Organization

This file describes the documentation structure for the SwarmCracker project.

## Root Documentation

These files live in the project root and provide essential project information:

- **README.md** - Main project overview, features, quick start guide, and CLI reference
- **AGENTS.md** - Project guide for AI agents and human contributors (setup, architecture, workflows)
- **CONTRIBUTING.md** - Contribution guidelines and code of conduct
- **PROJECT.md** - Project status, roadmap, and progress tracking
- **LICENSE** - Apache 2.0 license

## Detailed Documentation (`docs/`)

In-depth guides and technical documentation:

- **ARCHITECTURE.md** - System design, components, data flow, and deployment architecture
- **INSTALL.md** - Installation instructions for various environments
- **CONFIG.md** - Configuration reference with examples
- **TESTING.md** - Testing guide, running tests, writing tests
- **DEVELOPMENT.md** - Development workflow, best practices, debugging

## Test Reports (`docs/reports/`)

Historical test coverage reports and implementation summaries:

- **IMAGE_PREPARER_TESTS_REPORT.md** - Image preparer test coverage and implementation
- **NETWORK_MANAGER_TESTS_REPORT.md** - Network manager test coverage and implementation

## Quick Reference

- **README.md** contains the complete CLI reference
- **docs/INSTALL.md** contains usage examples
- **docs/CONFIG.md** contains configuration examples
